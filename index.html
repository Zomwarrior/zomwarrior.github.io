Computer Graphics Panorama Generator - Feb 2025

Panorama Generation: Developed a MATLAB-based program to automatically generate panoramic images from multiple input images. The system used keypoints and image stitching techniques to align and merge images into a seamless panorama.

Keypoint Detection & Feature Matching: Utilized SURF (Speeded-Up Robust Features) for keypoint detection and feature extraction across images. The system applied RANSAC (Random Sample Consensus) for robust homography estimation and feature matching between overlapping image pairs.

Stitching Process: Implemented both horizontal and vertical image stitching, aligning multiple images from different sets. The program handled issues like image warping, blending, and feathering to ensure a smooth transition between the stitched images.

Automation & Output: The tool automated the entire panorama generation process, handling multiple image sets, processing keypoints, computing transformations, and stitching images together. It saved the final panoramic images in a specified output path.

Technologies: The project was implemented in MATLAB, using advanced image processing techniques like SURF, RANSAC, and homography transformations. The project utilized custom functions to handle image stitching, blending, and keypoint extraction efficiently.
